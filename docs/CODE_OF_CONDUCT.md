# Code of Conduct
## Purpose

This is a personal, fun, and educational project. The goal is to learn, experiment, 
and build things because they’re interesting—not because everything must be perfect.

Everyone interacting with this project is expected to keep things respectful and constructive.

## Guidelines

- Be respectful and polite. Critique ideas, code, or design—not people.
- Keep feedback constructive and relevant.
- Remember this project is maintained in free time; suggestions may not always be accepted.
- Harassment, hateful language, personal attacks, trolling, spam, or demanding features or deadlines are not allowed.

Let’s keep this project fun and collaborative.

## Maintainer Rights

This project is maintained by a single author.

The maintainer has full discretion to:
- Accept or reject issues, pull requests, or suggestions.
- Edit or remove comments or contributions.
- Block or ban users who violate this Code of Conduct.
- No justification is required beyond maintaining a healthy and enjoyable project.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues and pull requests
- Discussions and comments
- Any interaction directly related to this project

## Final Note

This project is shared in good faith. If you’re here to learn, experiment, or help—welcome.
If you’re here to argue, demand, or create friction—this may not be the right place.

Let’s keep it simple and enjoyable.
